# English-apps

Gerunds practice app built with React, Vite, and Firebase initialization.

## Features

- Classifies gerunds by sentence role
- Includes reading practice with pronunciation guidance
- Matches gerund-focused words and verb phrases to example sentences
- Provides interactive classification exercises

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
```
